Python Strings in Depth

Strings are one of the most commonly used data types in Python. In this chapter, we'll dive deep into various string operations and common methods.

Creating Strings

In Python, strings can be created using single quotes, double quotes, or triple quotes:

# Single quote string str1 = 'Hello' # Double quote string str2 = "World" # Triple quote string (can span multiple lines) str3 = ''' This is a multi-line string '''

String Concatenation

Using + Operator

greeting = "Hello" name = "Python" message = greeting + " " + name print(message) # Hello Python

Using join() Method

words = ["Hello", "Beautiful", "World"] sentence = " ".join(words) print(sentence) # Hello Beautiful World

String Repetition

echo = "Ha " * 3 print(echo) # Ha Ha Ha # Create a separator line line = "-" * 50 print(line)

String Length

text = "Python Programming" print(len(text)) # 18
üí° Note: The len() function returns the number of characters in the string, including spaces and punctuation.

Accessing Characters

Strings are immutable sequences in Python. You can access individual characters by index:

word = "Python" print(word[0]) # P (first character) print(word[1]) # y print(word[-1]) # n (last character) print(word[-2]) # o

String Slicing

Use slicing to get a portion of the string:

text = "Hello Python" print(text[0:5]) # Hello print(text[6:]) # Python print(text[:5]) # Hello print(text[::2]) # HloPto (every other character) print(text[::-1]) # nohtyP olleH (reverse string)
üí° Slice Syntax: [start:end:step]. Start is included, end is excluded.

String Formatting

f-string (Python 3.6+)

name = "Alice" age = 25 # Using f-string (recommended) message = f"My name is {name}, I'm {age} years old" print(message) # Expression evaluation a, b = 10, 5 print(f"{a} + {b} = {a+b}") # 10 + 5 = 15 # Number formatting pi = 3.14159 print(f"œÄ = {pi:.2f}") # œÄ = 3.14

format() Method

# Positional arguments print("I'm {}, {} years old".format("Alice", 25)) # Keyword arguments print("I'm {name}, {age} years old".format(name="Alice", age=25)) # Number formatting print("Price: {:.2f} dollars".format(19.99)) # Price: 19.99 dollars

Common String Methods

Case Conversion

text = "hello python" print(text.upper()) # HELLO PYTHON print(text.lower()) # hello python print(text.title()) # Hello Python print(text.capitalize()) # Hello python

Search and Replace

text = "Hello Python, Python is great" # Find substring position print(text.find("Python")) # 6 print(text.index("Python")) # 6 print(text.rfind("Python")) # 15 (search from right) # Replace new_text = text.replace("Python", "Java") print(new_text) # Hello Java, Java is great

Removing Whitespace

text = " Hello Python " print(text.strip()) # "Hello Python" (remove both ends) print(text.lstrip()) # "Hello Python " (remove left) print(text.rstrip()) # " Hello Python" (remove right) # Remove specified characters text2 = "***Hello***" print(text2.strip("*")) # "Hello"

Split and Join

# Split string text = "apple,banana,orange" fruits = text.split(",") print(fruits) # ['apple', 'banana', 'orange'] # Split by whitespace sentence = "Hello World Python" words = sentence.split() print(words) # ['Hello', 'World', 'Python'] # Split into lines multiline = """Line 1 Line 2 Line 3""" lines = multiline.splitlines() print(lines) # ['Line 1', 'Line 2', 'Line 3']

Check Methods

print("hello".islower()) # True print("HELLO".isupper()) # True print("Hello123".isalnum()) # True (letters and numbers) print("Hello".isalpha()) # True (letters only) print("12345".isdigit()) # True (digits only) print(" ".isspace()) # True (whitespace only)

String Alignment

text = "Python" print(text.center(20, "*")) # *******Python******* print(text.ljust(20, "-")) # Python------------- print(text.rjust(20, "-")) -------------Python

Counting and Checking

text = "hello world hello" # Count substring occurrences print(text.count("hello")) # 2 print(text.count("l")) # 5 # Check start/end print(text.startswith("hello")) # True print(text.endswith("hello")) # True print(text.startswith("world")) # False

Escape Characters

# Common escape characters text = "Hello\nWorld" # Newline quote = "She said: \"Hi\"" # Quotes path = "C:\\Users\\Name" # Backslash tab = "Hello\tWorld" # Tab # Use raw string (no escaping) raw_path = r"C:\Users\Name" print(raw_path) # C:\Users\Name

Strings Are Immutable

‚ö†Ô∏è Important: Strings are immutable! You cannot directly modify characters in a string.
text = "Python" # text[0] = "J" # Error! Cannot modify directly # Correct approach: create new string new_text = "J" + text[1:] print(new_text) # Jython

String Methods Summary

Method Description
len(s) Return string length
s.upper() Convert to uppercase
s.lower() Convert to lowercase
s.title() Convert to title case
s.strip() Remove whitespace from both ends
s.split() Split string into list
s.join(iterable) Join sequence into string
s.find(sub) Find substring position
s.replace(old, new) Replace substring
s.startswith(prefix) Check if starts with prefix
s.endswith(suffix) Check if ends with suffix
s.count(sub) Count substring occurrences
