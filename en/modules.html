Python Modules in Depth

Modules are files containing Python definitions and statements. Using modules helps organize code, making it more maintainable and reusable.

What is a Module?

In Python, a module is a file containing all your defined functions and variables, with a .py extension.

üí° Why Use Modules?
  • Code Organization: Group related functionality together
  • Code Reuse: Use the same code in multiple programs
  • Namespaces: Avoid variable name conflicts
  • Maintainability: Easier to locate and fix issues

Importing Modules

Import Entire Module

import math # Use module name as prefix print(math.pi) # 3.141592653589793 print(math.sqrt(16)) # 4.0 print(math.pow(2, 3)) # 8.0

Import Module with Alias

import math as m print(m.pi) # 3.141592653589793 print(m.sqrt(16)) # 4.0

Import Specific Content

from math import pi, sqrt print(pi) # 3.141592653589793 (no prefix needed) print(sqrt(25)) # 5.0

Common Built-in Modules

random - Random Numbers

import random print(random.randint(1, 10)) # Random integer 1-10 print(random.random()) # Random float 0-1 print(random.choice(["a", "b", "c"])) # Random element # Shuffle list cards = ["A", "K", "Q", "J"] random.shuffle(cards) print(cards) # Shuffled order

datetime - Dates and Times

from datetime import date, datetime, timedelta # Current date today = date.today() print(today) # 2024-XX-XX # Current time now = datetime.now() print(now) # 2024-XX-XX XX:XX:XX.XXXXXX # Date calculation tomorrow = today + timedelta(days=1) print(tomorrow) # Tomorrow's date

json - JSON Data

import json # Python object to JSON string data = {"name": "Alice", "age": 25} json_str = json.dumps(data) print(json_str) # {"name": "Alice", "age": 25} # JSON string to Python object parsed = json.loads(json_str) print(parsed["name"]) # Alice
