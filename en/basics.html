Python Basic Syntax

Before learning Python, it's important to understand its basic syntax. Python is known for being simple and readable. Let's explore it together.

Python's Syntax Features

  • Simple and elegant: Code reads as naturally as English
  • Uses indentation: Uses indentation to represent code blocks, not braces
  • Case-sensitive: variable and Variable are two different variables
  • No semicolons needed: No semicolon needed at the end of each statement

Comments

Comments are used to explain code and are ignored by Python.

Single-line Comments

# This is a single-line comment print("Hello") # Comments can also be written after code

Multi-line Comments

''' This is a multi-line comment Can write multiple lines ''' """ This is also a multi-line comment Usually used for function or module documentation """

Indentation and Code Blocks

Python uses indentation to organize code blocks. Code at the same indentation level belongs to the same block.

‚ö†Ô∏è Important: Python is very strict about indentation! Must consistently use spaces or tabs, 4 spaces is recommended as indentation.
if 5 > 3: print("5 is greater than 3") # This is part of the if statement block print("This is also part of the if statement") print("This is not in the if statement")
5 is greater than 3 This is also part of the if statement This is not in the if statement

Variables

Variables are containers for storing data. In Python, you don't need to declare variable types.

# Create variables name = "Alice" age = 25 height = 1.75 is_student = True # Use variables print("Name:", name) print("Age:", age) print("Height:", height) print("Is student:", is_student)

Variable Naming Rules

  • Variable names must start with a letter or underscore
  • Variable names can only contain letters, numbers, and underscores
  • Variable names are case-sensitive
  • Cannot use Python keywords as variable names
  • Variable names should be meaningful and self-descriptive
# Legal variable names my_name = "John" _user_id = 123 age2 = 30 # Illegal variable names # 2name = "Jane" # Error: Cannot start with a number # my-name = "Bob" # Error: Cannot contain hyphens # class = "Tom" # Error: class is a keyword

Data Types

Python has several basic data types:

Integers (int)

a = 10 b = -5 c = 0 d = 1_000_000 # Large numbers can use underscores as separators print(a, b, c, d)

Floats (float)

pi = 3.14159 temperature = -10.5 scientific = 1.23e-5 # Scientific notation print(pi, temperature, scientific)

Strings (str)

name1 = 'Single quote string' name2 = "Double quote string" name3 = """Triple quote string Can span multiple lines""" print(name1) print(name2) print(name3)

Booleans (bool)

is_true = True is_false = False print(is_true, is_false)

Basic Operations

Arithmetic Operators

a = 10 b = 3 print("Addition:", a + b) # 13 print("Subtraction:", a - b) # 7 print("Multiplication:", a * b) # 30 print("Division:", a / b) # 3.3333... print("Floor division:", a // b) # 3 print("Modulus:", a % b) # 1 print("Exponentiation:", a ** b) # 1000

Comparison Operators

a = 10 b = 5 print("a == b:", a == b) # False print("a != b:", a != b) # True print("a > b:", a > b) # True print("a < b:", a < b) # False print("a >= b:", a >= b) # True print("a <= b:", a <= b) # False

Logical Operators

a = True b = False print("a and b:", a and b) # False print("a or b:", a or b) # True print("not a:", not a) # False

String Operations

String Concatenation

first_name = "John" last_name = "Smith" full_name = first_name + last_name print(full_name) # JohnSmith

String Repetition

echo = "Hello " * 3 print(echo) # Hello Hello Hello

String Formatting

name = "Alice" age = 25 # Method 1: Using f-string (Python 3.6+) print(f"My name is {name}, I'm {age} years old") # Method 2: Using format() print("My name is {}, I'm {} years old".format(name, age)) # Method 3: Using % operator print("My name is %s, I'm %d years old" % (name, age))
My name is Alice, I'm 25 years old My name is Alice, I'm 25 years old My name is Alice, I'm 25 years old

Type Conversion

Sometimes we need to convert one type to another:

# Convert to integer x = int("123") print(x) # 123 print(type(x)) # # Convert to float y = float("3.14") print(y) # 3.14 # Convert to string z = str(123) print(z) # "123" # Convert to boolean print(bool(1)) # True print(bool(0)) # False print(bool("")) # False print(bool("abc")) # True

Getting User Input

Use the input() function to get user keyboard input:

name = input("Please enter your name: ") print("Hello, " + name + "!")

age = int(input("Please enter your age: ")) print("Next year you will be " + str(age + 1) + " years old")
