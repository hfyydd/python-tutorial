Data Types in Detail

Python has multiple built-in data types. Understanding these data types is crucial for writing efficient and correct Python programs.

Numeric Types

Integers (int)

Integers can be positive, negative, or zero, with no size limit.

# Integer examples a = 10 b = -5 c = 0 big_number = 999999999999999999999999999 # Base representations binary = 0b1010 # Binary 10 octal = 0o12 # Octal 10 hexadecimal = 0xA # Hexadecimal 10 print(binary, octal, hexadecimal) # 10 10 10

Floats (float)

Floats are numbers with decimal points.

# Float examples a = 3.14 b = -0.001 c = 1.23e5 # Scientific notation: 123000.0 d = 1.23e-2 # Scientific notation: 0.0123 print(a, b, c, d)
‚ö†Ô∏è Note: Float calculations may have precision issues:
print(0.1 + 0.2) # Output: 0.30000000000000004

Complex Numbers (complex)

Python also supports complex number operations.

z = 3 + 4j # j represents the imaginary unit print(z.real) # Real part: 3.0 print(z.imag) # Imaginary part: 4.0

Boolean Type

The boolean type has only two values: True and False.

is_sunny = True is_raining = False # Boolean operations print(is_sunny and is_raining) # False print(is_sunny or is_raining) # True print(not is_sunny) # False
üí° Truth Value Testing: In Python, the following values are considered "False":
  • False
  • None
  • Zero numbers (0, 0.0, 0j)
  • Empty sequences ('', (), [])
  • Empty mappings ({})
All other values are considered "True".

Strings (str)

Strings are text enclosed in single quotes, double quotes, or triple quotes.

# Single and double quotes s1 = 'Hello' s2 = "World" # Triple quotes can span multiple lines s3 = ''' This is a multi-line string ''' # String concatenation greeting = s1 + " " + s2 print(greeting) # Hello World # String repetition laugh = "Ha" * 3 print(laugh) # HaHaHa

String Indexing and Slicing

s = "Python" # Indexing (starts from 0) print(s[0]) # P print(s[-1]) # n (last character) # Slicing [start:end:step] print(s[0:2]) # Py print(s[2:5]) # tho print(s[:3]) # Pyt (from start to index 3) print(s[3:]) # hon (from index 3 to end) print(s[::2]) # Pto (every other character) print(s[::-1]) # nohtyP (reverse string)

String Methods

s = " Hello World " # Case conversion print(s.upper()) # " HELLO WORLD " print(s.lower()) # " hello world " print(s.strip()) # "Hello World" (remove leading/trailing spaces) # Searching print(s.find("World")) # 8 (returns -1 if not found) print("World" in s) # True # Replacing print(s.replace("World", "Python")) # " Hello Python " # Splitting print(s.split()) # ["Hello", "World"]

Lists (list)

Lists are mutable ordered collections.

# Create lists fruits = ["apple", "banana", "orange"] numbers = [1, 2, 3, 4, 5] mixed = ["text", 42, True, 3.14] # Access elements print(fruits[0]) # "apple" print(fruits[-1]) # "orange" # Modify elements fruits[1] = "grape" print(fruits) # ["apple", "grape", "orange"] # Add elements fruits.append("watermelon") print(fruits) # ["apple", "grape", "orange", "watermelon"] # Remove elements fruits.remove("orange") print(fruits) # ["apple", "grape", "watermelon"] # Get length print(len(fruits)) # 3

List Slicing

nums = [0, 1, 2, 3, 4, 5] print(nums[1:4]) # [1, 2, 3] print(nums[:3]) # [0, 1, 2] print(nums[3:]) # [3, 4, 5] print(nums[::2]) # [0, 2, 4]

Tuples (tuple)

Tuples are immutable ordered collections, represented with parentheses.

# Create tuples point = (3, 4) colors = ("red", "green", "blue") # Access elements (same as lists) print(point[0]) # 3 # Tuples cannot be modified # point[0] = 5 # This will raise an error! # Single-element tuples need a comma single = (42,) # Note the comma
üí° When to use tuples?
  • When data should not be modified
  • When using as dictionary keys
  • When returning multiple values from functions

Dictionaries (dict)

Dictionaries are collections of key-value pairs.

# Create dictionary person = { "name": "John Smith", "age": 25, "city": "Beijing" } # Access values print(person["name"]) # "John Smith" print(person.get("age")) # 25 # Add/modify key-value pairs person["email"] = "test@example.com" person["age"] = 26 # Delete key-value pairs del person["city"] # Iterate through dictionary for key, value in person.items(): print(f"{key}: {value}")

Sets (set)

Sets are unordered collections of unique elements.

# Create sets s1 = {1, 2, 3, 4} s2 = {3, 4, 5, 6} # Set operations print(s1 | s2) # Union: {1, 2, 3, 4, 5, 6} print(s1 & s2) # Intersection: {3, 4} print(s1 - s2) # Difference: {1, 2} # Add/remove elements s1.add(5) s1.remove(2)

None Type

None represents "no value" or "empty value".

result = None if result is None: print("Result is empty")

Type Checking and Conversion

# Type checking x = "Hello" print(type(x)) # print(isinstance(x, str)) # True # Type conversion print(int("123")) # 123 print(float("3.14")) # 3.14 print(str(123)) # "123" print(bool(1)) # True print(list("abc")) # ["a", "b", "c"] print(tuple([1, 2])) # (1, 2)
‚úÖ Summary
  • Integers (int), floats (float), complex numbers (complex) are numeric types
  • Strings (str) are used for text processing
  • Lists (list) are mutable, tuples (tuple) are immutable
  • Dictionaries (dict) store key-value pairs
  • Sets (set) store unique elements
  • None represents empty value
