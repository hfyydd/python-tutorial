Python Functions in Depth

Functions are the basic way to organize code. They encapsulate code blocks for reuse, making code clearer and easier to maintain.

What is a Function?

A function is a reusable block of code that takes input (parameters), processes it, and returns output (return value).

üí° Why Use Functions?
  • Code Reuse: Avoid writing the same code repeatedly
  • Improve Readability: Break complex problems into smaller ones
  • Easier Maintenance: Changes only need to be made in one place
  • Modularity: Make code structure clearer

Defining Functions

In Python, use the def keyword to define a function:

def greet(): """This is a simple greeting function""" print("Hello, Python!") # Call the function greet()
Hello, Python!

Function Parameters

Positional Parameters

def greet_user(name): print(f"Hello, {name}!") greet_user("Alice") greet_user("Bob")
Hello, Alice! Hello, Bob!

Multiple Parameters

def greet(name, greeting): print(f"{greeting}, {name}!") greet("Alice", "Hello") greet("Bob", "Good morning")

Default Parameters

def greet(name, greeting="Hello"): print(f"{greeting}, {name}!") greet("Alice") # Use default greeting greet("Bob", "Good morning") # Custom greeting
‚ö†Ô∏è Note: Default parameters must come after non-default parameters!

Keyword Arguments

def describe_pet(animal, name): print(f"I have a {animal} named {name}") # Use keyword arguments, order can be changed describe_pet(animal="cat", name="Whiskers") describe_pet(name="Fido", animal="dog")

Return Values

Functions can use the return statement to return values:

def add(a, b): return a + b result = add(5, 3) print(result) # 8 # Can also be used directly in expressions print(add(10, 20)) # 30

Returning Multiple Values

Python functions can return multiple values (actually returns a tuple):

def get_name(): return "Lee", "Ming" first, last = get_name() print(first) # Lee print(last) # Ming # Can also receive as a tuple name = get_name() print(name) # ('Lee', 'Ming')

Function Docstrings

Use triple-quoted strings to add documentation to functions:

def calculate_area(radius): """ Calculate the area of a circle Args: radius (float): The radius of the circle Returns: float: The area of the circle """ import math return math.pi * radius ** 2 # View function documentation print(calculate_area.__doc__)

Variable Scope

Local Variables

def my_function(): local_var = "I am a local variable" print(local_var) my_function() # Output: I am a local variable # print(local_var) # Error! Cannot access local variable from outside

Global Variables

global_var = "I am a global variable" def my_function(): print(global_var) # Can access global variable my_function() # Output: I am a global variable print(global_var) # Output: I am a global variable

Modifying Global Variables

counter = 0 def increment(): global counter # Declare using global variable counter += 1 print(counter) increment() # 1 increment() # 2 print(counter) # 2

Lambda Functions

Lambda functions are anonymous small functions created with the lambda keyword:

# Regular function def add(x, y): return x + y # Equivalent lambda function add_lambda = lambda x, y: x + y print(add(3, 5)) # 8 print(add_lambda(3, 5)) # 8 # Commonly used for short functions numbers = [1, 5, 3, 9, 2] sorted_numbers = sorted(numbers, key=lambda x: -x) print(sorted_numbers) # [9, 5, 3, 2, 1]

Recursive Functions

Recursive functions call themselves. Here's an example of calculating factorial:

def factorial(n): """Calculate factorial of n""" if n <= 1: # Base case return 1 return n * factorial(n - 1) # Recursive call print(factorial(5)) # 120 print(factorial(10)) # 3628800
