Installing Python

Before you start learning Python, you need to install the Python environment on your computer. Don't worry, this process is very simple!

‚ö†Ô∏è Important Note: Please make sure to install Python 3.x version (3.10 or higher is recommended). Python 2.x is no longer maintained and is not recommended for use.

Check if Python is Already Installed

First, let's check if your computer already has Python installed:

1Open Terminal (Mac/Linux) or Command Prompt (Windows)

2Enter the following command:

python --version

3Or try:

python3 --version

If you see output similar to Python 3.10.6, congratulations! Python is already installed! You can skip directly to the next chapter.

If you see "command not found" or a similar message, continue with the installation steps below.

Windows Installation

1Visit the Python official download page: python.org/downloads

2Click the "Download Python" button to download the latest version

3Run the downloaded installer

üì∑ Installation interface screenshot
(Check the "Add Python to PATH" option)

4Important: Make sure to check "Add Python to PATH" in the installation interface

5Click "Install Now" to start the installation

6Wait for the installation to complete, then click "Close"

‚úÖ Verify Installation: Open Command Prompt and type python --version. If you see the version number, the installation was successful.

Mac Installation

1Mac systems usually come with Python, but the version may be outdated

2It's recommended to use Homebrew to install the latest version of Python:

# If you don't have Homebrew installed, install it first /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # Install Python using Homebrew brew install python

3Or download the installer from the official website: python.org/downloads

4Download the macOS installer and double-click to run the installation

Linux Installation

1Most Linux distributions come with Python

2Ubuntu/Debian systems:

sudo apt update sudo apt install python3 python3-pip

3Fedora systems:

sudo dnf install python3 python3-pip

4Arch Linux:

sudo pacman -S python python-pip

Verify Installation

After installation, let's verify that Python is correctly installed:

1Open Terminal or Command Prompt

2Enter the following command to enter the Python interactive environment:

python

or

python3

3If you see a prompt similar to the following, the installation was successful:

Python 3.10.6 (main, Nov 14 2022) [GCC 11.3.0] on linux Type "help", "copyright", "credits" or "license" for more information. >>>

4Type exit() or press Ctrl+D to exit the Python environment

Configure Python Environment

Installing Packages with pip

pip is Python's package management tool for installing third-party libraries. Let's verify that pip is available:

pip --version # or pip3 --version

If the pip version number is displayed, pip is correctly installed.

Upgrade pip (Recommended)

python -m pip install --upgrade pip

Configure Domestic Mirror Sources (Optional)

If download speeds are slow, you can configure domestic mirror sources:

# Tsinghua source pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple # Alibaba source pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/

Choose a Code Editor

While Python comes with IDLE for writing code, it's recommended to use a more professional editor:

üîß Recommended Editors:

  • Visual Studio Code: Free, powerful, with rich plugins (highly recommended)
  • PyCharm Community: Professional Python IDE, free version has sufficient features
  • Sublime Text: Lightweight, fast startup
  • Jupyter Notebook: Great for data analysis and learning

Installing Visual Studio Code

  1. Visit code.visualstudio.com
  2. Download the installer for your system
  3. Open VS Code after installation
  4. Click the Extensions icon on the left and search for "Python"
  5. Install the official Microsoft Python extension
