Conditional Statements

Conditional statements allow programs to execute different code based on different situations. This is one of the most fundamental and important control structures in programming.

if Statement

The simplest conditional statement is the if statement:

age = 18 if age >= 18: print("You are an adult")
You are an adult

if-else Statement

If you need to execute another block of code when the condition is not met, use else:

age = 16 if age >= 18: print("You are an adult") else: print("You are not an adult yet")
You are not an adult yet

if-elif-else Statement

When multiple conditions need to be checked, use elif (short for "else if"):

score = 85 if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" elif score >= 60: grade = "D" else: grade = "F" print(f"Your grade is: {grade}")
Your grade is: B
üí° Note: You can have multiple elif statements, but only one else (which can also be omitted). Python checks conditions from top to bottom. Once a condition is True, it executes the corresponding block and skips the remaining condition checks.

Comparison Operators

Comparison operators commonly used in conditional statements:

a = 10 b = 5 print(a == b) # Equal print(a != b) # Not equal print(a > b) # Greater than print(a < b) # Less than print(a >= b) # Greater than or equal print(a <= b) # Less than or equal

Logical Operators

Use logical operators to combine multiple conditions:

age = 25 has_ticket = True # and: Both conditions must be True for the result to be True if age >= 18 and has_ticket: print("You can enter the cinema") # or: Any condition being True makes the result True is_weekend = False if is_weekend or age < 12: print("You can enjoy the discount") # not: Negation if not has_ticket: print("You need to buy a ticket first")

Membership Operators

fruits = ["apple", "banana", "orange"] # in: Check if contained if "apple" in fruits: print("We have apples") # not in: Check if not contained if "watermelon" not in fruits: print("We don't have watermelon")

Identity Operators

a = [1, 2, 3] b = a c = [1, 2, 3] # is: Check if it's the same object print(a is b) # True (b and a point to the same object) print(a is c) # False (c is a different object) print(a == c) # True (values are equal)

Nested Conditions

Conditional statements can be nested:

age = 20 has_license = True if age >= 18: if has_license: print("You can drive") else: print("You need to get a license first") else: print("You are not of legal age yet")
‚ö†Ô∏è Note: Too many nesting levels reduces code readability. Consider using logical operators to simplify.

Ternary Operator

Python has a concise ternary operator (conditional expression):

age = 20 # Traditional way if age >= 18: status = "adult" else: status = False # Ternary operator way status = "adult" if age >= 18 else "minor" print(status)

Common Use Cases

1. User Input Validation

password = input("Please enter a password: ") if len(password) < 6: print("Password is too short!") elif len(password) > 20: print("Password is too long!") else: print("Password length is appropriate")

2. Number Range Checking

temperature = 25 if temperature < 0: print("Very cold") elif temperature < 15: print("A bit cold") elif temperature < 25: print("Comfortable temperature") elif temperature < 35: print("A bit hot") else: print("Very hot")

3. Odd/Even Number Check

num = 7 if num % 2 == 0: print(f"{num} is even") else: print(f"{num} is odd")
