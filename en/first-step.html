Your First Python Program

Congratulations on making it here! Following programming tradition, our first program will be "Hello, World!" ‚Äî a greeting to the world.

Using the Python Interactive Environment

Python comes with an interactive environment (REPL) that's perfect for quickly testing code and learning:

Step 1: Open Terminal or Command Prompt

Step 2: Enter the following command:

python

or

python3

Step 3: You'll see a prompt similar to:

Python 3.10.6 (main, Jan 1 2024) [GCC] on linux Type "help", "copyright", "credits" or "license" for more information. >>>

The >>> is Python's prompt, indicating Python is waiting for your input.

Your First Line of Code

Now, let's enter our first line of Python code:

print("Hello, World!")

Press Enter and you'll see:

Hello, World!
üéâ Congratulations! You just wrote your first Python program!

What Does This Code Mean?

Let's analyze this line of code:

  • print() is a function that outputs content to the screen
  • "Hello, World!" is a string, enclosed in double quotes
  • The entire line means: call the print function to output the string "Hello, World!"

Keep Experimenting

In the interactive environment, try more commands:

>>> print("Python is really simple!") Python is really simple! >>> print(1 + 1) 2 >>> print("1 + 1 =", 1 + 1) 1 + 1 = 2 >>> name = "Alice" >>> print("Hello, " + name) Hello, Alice

You can keep practicing in the interactive environment. Type exit() or press Ctrl+D to exit.

Creating a Python File

Besides the interactive environment, we usually write code in files. Let's create a Python file:

Step 1: Open your code editor (like VS Code)

Step 2: Create a new file and save it as hello.py

Step 3: Enter the following code:

# My first Python program print("Hello, World!") print("Welcome to the world of Python!") # Simple math calculations print("1 + 1 =", 1 + 1) print("10 - 3 =", 10 - 3) # String concatenation name = "Python Learner" print("Hello, " + name)

Step 4: Save the file

Step 5: In Terminal, navigate to the file's directory and run:

python hello.py

or

python3 hello.py

You should see the following output:

Hello, World! Welcome to the world of Python! 1 + 1 = 2 10 - 3 = 7 Hello, Python Learner
‚úÖ Success! You've created your first Python script file!

About Comments

You may have noticed some lines starting with # in the code. These are comments:

# This is a single-line comment # Comments are not executed by Python # They are used to explain code print("This line will be executed") # Comments can also be written after code

Comments are for programmers to read; Python ignores them. Writing good comments is a good habit!

The print() Function in Detail

print() is one of Python's most commonly used functions. Let's learn more about it:

Printing Multiple Items

print("Hello", "World", "!")
Hello World !

By default, print separates multiple arguments with spaces.

Using sep Parameter to Customize Separator

print("Hello", "World", "!", sep="-")
Hello-World-!

Using end Parameter to Control Ending

print("Hello", end=" ") print("World")
Hello World

By default, print adds a newline after output. Using the end parameter can change this behavior.

Common Errors

Some common mistakes beginners make:

Error 1: Forgetting quotes
print(Hello) # Error! Hello will be treated as a variable name
Error 2: Mismatched parentheses
print("Hello" # Error! Missing closing parenthesis
Error 3: Using Chinese punctuation
print("Hello"); # Error! Should use English semicolon or no semicolon
