What is Python?

Python is a computer programming language. You may have heard of many popular programming languages:

  • C - very difficult to learn
  • Java - very popular
  • Basic - suitable for beginners
  • JavaScript - suitable for web programming

So what kind of language is Python?

First, let's cover some basics of programming languages. Using any programming language to develop programs is about getting the computer to do work, like downloading an MP3, writing a document, etc. The CPU that does the work only understands machine instructions, so despite the vast differences between programming languages, they all must be "translated" into machine instructions that the CPU can execute.

üí° Code Comparison: To complete the same task, C might require 1000 lines of code, Java 100 lines, while Python might only need 20 lines.

So Python is a very high-level language. You might ask, is less code bad?

The cost of less code is slower execution speed. A C program might run in 1 second, Java in 2 seconds, while Python might take 10 seconds.
