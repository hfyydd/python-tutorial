Loops

Loops allow programs to repeatedly execute a block of code. Python has two main loop structures: for loops and while loops.

for Loops

for loops are used to iterate over sequences (like lists, strings, tuples) or other iterable objects.

Iterating Over Lists

fruits = ["apple", "banana", "orange"] for fruit in fruits: print(fruit)
apple banana orange

Iterating Over Strings

for char in "Python": print(char)

Using the range() Function

The range() function generates a sequence of integers:

# range(stop): 0 to stop-1 for i in range(5): print(i) # Output: 0, 1, 2, 3, 4 # range(start, stop): start to stop-1 for i in range(1, 6): print(i) # Output: 1, 2, 3, 4, 5 # range(start, stop, step): specify step size for i in range(0, 10, 2): print(i) # Output: 0, 2, 4, 6, 8

Iterating Over Dictionaries

person = { "name": "John Smith", "age": 25, "city": "Beijing" } # Iterate over keys for key in person: print(key) # Iterate over key-value pairs for key, value in person.items(): print(f"{key}: {value}")

while Loops

while loops repeatedly execute a block of code while the condition is True:

count = 0 while count < 5: print(count) count += 1
0 1 2 3 4
‚ö†Ô∏è Note: When using while loops, make sure the loop condition will eventually become False, otherwise you'll create an infinite loop!

Loop Control Statements

break: Exit the Loop

for i in range(10): if i == 5: break # Exit loop when i equals 5 print(i) # Output: 0, 1, 2, 3, 4

continue: Skip This Iteration

for i in range(5): if i == 2: continue # Skip this iteration when i equals 2 print(i) # Output: 0, 1, 3, 4

else Clause

Loops can have an else clause that executes when the loop finishes normally (not interrupted by break):

# Normal completion, execute else for i in range(5): print(i) else: print("Loop completed normally") # Interrupted by break, don't execute else for i in range(5): if i == 3: break print(i) else: print("This won't be executed")

Nested Loops

# Print multiplication table for i in range(1, 10): for j in range(1, i + 1): print(f"{j}x{i}={i*j}", end="\t") print() # New line

Common Use Cases

1. Calculate Sum

total = 0 for i in range(1, 101): total += i print(f"The sum of 1 to 100 is: {total}")

2. Find Maximum Value in List

numbers = [23, 45, 12, 67, 34, 89] max_num = numbers[0] for num in numbers: if num > max_num: max_num = num print(f"The maximum value is: {max_num}")

List Comprehensions

List comprehensions provide a concise way to create lists:

# Traditional way squares = [] for i in range(10): squares.append(i ** 2) # List comprehension squares = [i ** 2 for i in range(10)] # List comprehension with condition even_squares = [i ** 2 for i in range(10) if i % 2 == 0] print(squares) # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81] print(even_squares) # [0, 4, 16, 36, 64]
