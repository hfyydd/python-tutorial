最初のPythonプログラム

ここまで来ておめでとうございます！プログラミングの伝統に従い、最初に書くプログラムは「Hello, World!」——世界に挨拶することです。

🎯 本章の目標

  • Python対話モードの使い方を学ぶ
  • 最初のPythonプログラムを書く
  • Pythonの基本的な出力方法を理解する

Python対話モードの使用

Pythonには対話モード（REPL）が付属しており、コードの迅速なテストや学習に最適です：

ステップ1：ターミナルまたはコマンドプロンプトを開く

ステップ2：以下のコマンドを入力します：

python

または

python3

ステップ3：以下のようなプロンプトが表示されます：

Python 3.10.6 (main, Jan 1 2024) [GCC] on linux Type "help", "copyright", "credits" or "license" for more information. >>>

この >>> はPythonのプロンプトで、Pythonが入力を待っていることを示します。

最初のコード行

それでは、最初のPythonコードを入力してみましょう：

print("Hello, World!")

Enterキーを押すと、以下が表示されます：

Hello, World!
🎉 おめでとうございます！ 最初のPythonプログラムを書きました！

このコードの意味は？

このコード行を分析してみましょう：

  • print() は画面に出力するための関数です
  • "Hello, World!" は二重引用符で囲まれた文字列です
  • コード行全体の意味：print関数を呼び出し、「Hello, World!」という文字列を出力する

続きを試す

対話モードで、さらに多くのコマンドを試してみましょう：

>>> print("Pythonはとても簡単です！") Pythonはとても簡単です！ >>> print(1 + 1) 2 >>> print("1 + 1 =", 1 + 1) 1 + 1 = 2 >>> name = "太郎" >>> print("こんにちは、" + name) こんにちは、太郎

対話モードで練習を続けることができます。exit()と入力するか、Ctrl+Dを押して終了します。

Pythonファイルの作成

対話モード以外にも、通常はコードをファイルに書きます。Pythonファイルを作成してみましょう：

ステップ1：コードエディタ（VS Codeなど）を開く

ステップ2：新しいファイルを作成し、hello.pyとして保存

ステップ3：以下のコードを入力します：

# 私の最初のPythonプログラム print("Hello, World!") print("Pythonの世界へようこそ！") # 簡単な計算 print("1 + 1 =", 1 + 1) print("10 - 3 =", 10 - 3) # 文字列の連結 name = "Python学習者" print("こんにちは、" + name)

ステップ4：ファイルを保存

ステップ5：ターミナルでファイルがあるディレクトリに移動し、実行します：

python hello.py

または

python3 hello.py

以下の出力が表示されます：

Hello, World! Pythonの世界へようこそ！ 1 + 1 = 2 10 - 3 = 7 こんにちは、Python学習者
✅ 成功！ 最初のPythonスクリプトファイルを作成しました！

コメントについて

コードに #で始まる行があることに気づいたかもしれません。これらはコメントです：

# これは一行コメントです # コメントはPythonによって実行されません # コードを説明するために使用されます print("この行は実行されます") # コメントはコードの後にも書けます

コメントはプログラマー向けのもので、Pythonはそれらを無視します。コメントを書くことは良い習慣です！

print()関数の詳細

print() はPythonで最もよく使われる関数の一つです。詳しく見ていきましょう：

複数の内容を出力

print("Hello", "World", "!")
Hello World !

デフォルトでは、printはスペースで区切って複数の引数を出力します。

sepパラメータで区切り文字をカスタマイズ

print("Hello", "World", "!", sep="-")
Hello-World-!

endパラメータで末尾を制御

print("Hello", end=" ") print("World")
Hello World

デフォルトでは、printは出力後に改行を追加します。endパラメータでこの動作を変更できます。

練習問題に挑戦

それでは、以下のプログラムを書いてみましょう：

練習問題：
  1. 名前と年齢を表示する
  2. 簡単な計算式と結果を表示する
  3. アスタリスクで図形（三角形など）を表示する

解答例：

# 個人情報を表示 print("名前：山田太郎") print("年齢：25") # 計算式を表示 print("5 * 6 =", 5 * 6) # アスタリスクの三角形を表示 print(" * ") print(" *** ") print(" ***** ") print("*******")

よくあるエラー

初心者がよく犯す間違い：

エラー1：引用符を忘れる
print(Hello) # エラー！Helloが変数名として扱われます
エラー2：括弧が一致しない
print("Hello" # エラー！閉じ括弧がありません
エラー3：日本語の句読点を使用
print("Hello")； # エラー！英語のセミコロンを使うか、セミコロンなしにします
